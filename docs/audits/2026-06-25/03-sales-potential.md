## 3.2 Scenario verdicts

### Warm traffic / referral / existing network — **HIGH potential**

- Buyer already trusts Norbert or arrives from zzpackage/game/portal.
- Live systems + honest PARTIAL labels increase trust vs agency theatre.
- €290 discovery is well explained; wizard is try-before-buy proof.
- **Recommended motion:** Direct to `/book-discovery/` or WhatsApp with owner-ecosystem + sales-funnel CS.

### Cold NL SMB (LinkedIn, ads, SEO) — **MEDIUM potential**

- Positioning and design pass the “product company” bar.
- Cold buyers need **motion proof** (wizard 45s, inbox 60s) — currently missing.
- Solutions pages for Wizard/Web Upgrade undersell vs results CS (generic copy).
- Field reports help but need Commander-verified numbers in `proof.ts`.
- **Blockers before scale:** BL-03 video (min. wizard), BL-02 (2–3 metrics), solutions SSoT (E-8).

### Investor / angel — **MEDIUM-LOW until BL-02**

- LOS + governance + Agent OS narrative is differentiated.
- Honesty framework (LIVE/PARTIAL/PLANNED) is investor-friendly.
- Commercial traction template in `proof-rules.md` still UNKNOWN.
- **Ready for:** technical diligence conversation; **not ready for:** traction slide without Commander data.

## 3.3 Competitive moat (what sells)

1. **End-to-end system story** — not “I build websites” or “I sell chatbots”.
2. **Production deployment** — same stack runs FlexGrafik print business.
3. **Human-in-the-loop** — reduces AI-fatigue objection for NL SMB.
4. **Wizard Cash Engine** — tangible revenue layer competitors rarely show live.
5. **Clinical qualification** — protects architect time; signals premium positioning.

## 3.5 Honest bottom line

**Can the site sell €290 discovery and €1,200+ projects?**
