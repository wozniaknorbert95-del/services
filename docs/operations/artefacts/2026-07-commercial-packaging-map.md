## Offer detail

### Automation Map
- **Sells:** diagnosis → fit → fixed-scope proposal or no-build  
- **Does not sell:** implementation hours  
- **Post-Map expectation copy required:** deliverable list, decision point, timing, no-build/referral  

### Sales Funnel / Wizard (+ Complex Quote & Design Intake)
- **Sells:** guided configure→price→checkout; optional INSPIRE intake→brief→Standard/Premium direction→wizard handoff  
- **Proof:** WIZ-* LIVE; INS-* PARTIAL badge + limitation (mockups ≠ print-ready; sales chat NOT GO)  
- **Demo:** `https://zzpackage.flexgrafik.nl/wizard/` · `…/voertuigreclame-ontwerp/`  
- **Forbidden:** uplift %, Gate D traction, autonomous consultant  

### Web Upgrade
- **Sells:** conversion-ready site, Wizard-first CTAs, qualification-ready architecture  
- **Proof:** PORT-01/02 LIVE; PORT-03 PARTIAL (no “qual agent live”)  
- **Add-on narrative:** supervised sales chat / hot-lead escalation  

### Lead Magnet Game
- **Sells:** selective engagement→reward→wizard bridge (fit-dependent vertical)  
- **Not:** default next step after every Web Upgrade  

### Inbox Killer
- **Sells:** classified lanes, draft replies, HITL send gate  
- **Proof:** process measurement — label test-environment honestly (142 msgs/scan)  
- **Boundary:** Quietforge B2B product ≠ entire jadzia-core  

### Operations Command Layer (named package, not separate route required)
- **Sells:** owner cockpit, lead/order/analytics spine, approval queue, weekly brief  
- **Proof:** JAD-01…04 LIVE  
- **Placement:** Ecosystem / Multi-System band + Jadzia case + Managed upsell  
- **Forbidden:** full autonomous COI
