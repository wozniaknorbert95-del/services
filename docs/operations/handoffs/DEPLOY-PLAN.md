## 2. Git commands

```bash
# 1. Update .gitignore to exclude dist/ (if not already)
echo "/dist/" >> .gitignore

# 2. Stage all source changes
git add src/ public/og/home.svg docs/ .gitignore

# 3. Stage deletions
git rm src/components/CTAFinal.tsx src/components/CTASection.tsx src/components/CaseStudies.tsx src/components/Hero.tsx src/components/HowItWorks.tsx src/components/PainPoints.tsx src/components/Products.tsx src/components/ProjectShowcase.tsx src/components/RiskReversal.tsx src/components/Solution.tsx src/components/Trust.tsx src/components/TrustBar.tsx

git commit -m "feat: Phase 0 & 1 — Quietforge rebrand + full Home rebuild

- Migrate design system to Quietforge (warm black + amber accent)

- Create 7 Home section components mapped to copy doc
```

## 4. If deploy fails

1. Check Vercel build logs for TypeScript errors
2. Run `npm run typecheck` + `npm run build` locally
3. Fix → commit → push
4. If `dist/` was accidentally committed, `git rm -r --cached dist/` + update `.gitignore`
