# LinkedIn Principles

---

## CO

Zasady prowadzenia **jednego** profilu LinkedIn jako kanału **Quietforge B2B**, z FlexGrafik wyłącznie jako **proof layer**.

**Zasada GTM-01 (nadrzędna):**

> LinkedIn nie sprzedaje druku — pokazuje system działający na FlexGrafik jako proof. CTA = Quietforge B2B.

---

## DLACZEGO

LinkedIn to jedyny kanał, gdzie budujesz **Conversion Systems Architect** dla NL SMB. Audyt (2026-06-29): positioning ~4.2/5, ale feed i ścieżka konwersji ~2/5 — bo treść mówi do inwestorów i founder pity, nie do kupującego Map.

---

## BO

Profil bez zasad = każdy post jest „wyjątkiem” (Post 4 funding, magnesy, flexgrafik.nl). Kupujący B2B nie wie, co kupić ani gdzie kliknąć.

## Zasady profilu (strategia, nie UI checklist)

### P1 — Tożsamość

| Zasada | Szczegół |
|--------|----------|
| Primary label | Conversion Systems Architect ([marketing-rules](../../canons/marketing-rules.md) MR-01) |
| Company | Quietforge |
| FlexGrafik | W About / Experience / proof — nie w headline jako „drukarnia” |
| Banner | Zachować Quietforge + 8-repo diagram (audyt: 5/5) |

### P2 — Konwersja

| Zasada | Docelowy kierunek | Wymaganie na stronie (homepage) |
|--------|-------------------|----------------------------------|
| Jedna akcja | Book Automation Map ([conversion-pipeline](../conversion-pipeline.md) L3) | Hero L3 + Featured card #1 = ten sam URL `/book-discovery/` |
| Featured (LI) | quietforge home, book-discovery, 1–2 case studies | **Mirror** homepage Featured strip — 3 cards: Map · /results/ · how-it-works |
| Link w poście | W **pierwszym komentarzu**, nie w body (zasięg) | Landing musi domknąć w 5 s: dual-brand + Problem→System→Effect |
| Pricing | Jawne odsyłanie do [marketing-strategy §8](../marketing-strategy.md) — nie „contact for pricing” | Homepage + /pricing/ = jedna macierz cen (audyt: drift P0) |
| CTA secondary | Live systems | `/results/` jako Featured card #2 — FlexGrafik proof bez consumer pitch |

### P3 — Feed

| Zasada | Szczegół |
|--------|----------|
| Język postów | EN |
| Struktura | Problem → System → Effect → CTA (komentarz) |
| FlexGrafik w poście | Screenshot / metryka / „in my print business” — nie oferta druku |
| Founder story | Dozwolone 20% mix ([05-content-pillars](./05-content-pillars.md)) — bez funding ask |
| Investor | [08-investor-track](./08-investor-track.md) — nie na głównym feedzie teraz |

### P4 — Honesty (LIVE / PARTIAL / DEMO)

Zgodnie z [proof-rules.md](../../canons/proof-rules.md):

| Mów | Nie mów |
|-----|---------|
| „Wizard checkout LIVE on zzpackage” | „System 80% complete” |
| „Jadzia COI Phase A+B LIVE; Procurement Brain PLANNED” | „Almost finished, 2 months left” |
| „161 SKUs” (site SSoT) | „200+ SKUs” jeśli Services nadal ma starą liczbę |
| PROVEN / DEMO / PLANNED labels | Fake testimonials, fake logos |

### P5 — Spójność z homepage (NOWE v2)

| Zasada | Szczegół |
|--------|----------|
| Post ↔ landing | Każdy post B2B z CTA w komentarzu musi **matchować** sekcję homepage, na którą prowadzi |
| Język | Problem→System→Effect w poście = ten sam beat w hero strony — **nie** repo dump w obu miejscach |
| Dual-brand | Post mówi „in my NL print business” → landing pokazuje **FlexGrafik jako proof**, nie druk |
| De-jargon | Jeśli post unika LangGraph/MCP/8-repo w hooku — homepage above fold też |
| Featured sync | LinkedIn Featured slots = homepage Featured cards (URL + kolejność) |
| Book Discovery | Copy Map €290 na LI = copy na `/book-discovery/` (audyt: pay&slot vs enquiry form — P0) |

**Test przed publikacją:** Otwórz URL z komentarza na mobile — czy w 5 s wiesz: (a) co sprzedaje Quietforge, (b) gdzie FlexGrafik dowodzi, (c) gdzie kliknąć Map?

## Szablon mentalny jednego posta

```
1. Hook — problem NL SMB (1–2 linie)
2. System — co zbudowałeś (bez repo dump)
3. Proof — 1 liczba lub screenshot + status LIVE/PARTIAL
4. Effect — co zmienia dla biznesu
5. CTA — w komentarzu: Book Automation Map + URL + UTM
```
